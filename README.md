"# eitango_ocr" 


azureを使った塾の復習教材作成業務効率化code

主に英語学習に活用予定

----は個人のをお使い下さい

ocr_eitango.py

入力画像を文字に変えてくれます。

tanngo.py

入力された文章が何単語で構成されているのか、また頻出単語とその回数を降順に表示します。

honnyaku.py

入力された文章を翻訳してくれます。

tanngo_translate.py

組み合わせ例＿＿頻出単語とその回数を降順に表示（日本語で翻訳付き）

tanngo_output.py

tanngotranslate.pyの出力を綺麗にしてみた（日本語はフォントの修正が必要）

csv_output.py

出力した単語リストをcsvファイルにデータとして落としてくれます。
